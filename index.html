mun.track functions much like a standard command line. Type the following commands in the textbox at the bottom of the page, and follow the relevant prompts.


helpDisplay this help dialog
settitleSet title
setquorumSet quorum
settimeSet speaking time
setextsSet speaking extensions
 
addAdd delegates to current speakers list
insertInsert delegate into current speakers list
changeChange delegate in current speakers list
flipFlip delegates in current speakers list
deleteRemove delegates from current speakerslist
nextAdvance speakers list
prevReturn to previous delegate in speakers list
 
switchSwitch between rolling/general/voting speakers list
voteEnter voting procedure
extendExtend current delegate's speaking time
timerStart large timer (for caucuses)
spacebarStart/stop speaking timer
